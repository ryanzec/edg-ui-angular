Please create view component code using modern angular 20 standards and best practices use the directory given (update any existing files and needed).

Guidelines:
- Always prefer Angular 20 modern standards and patterns that might be more "popular" but are for older versions of Angular
- Mostly utilize the set of component available via Angular Material 20 and customer components in `projects/shared-ui` library (formt he root of the repository)
- Should have minimal custom styling, pretty much limit to structural styling (spacing, size, etc.) of the element in this component itself
- If a change is needed in a component from the `projects/shared-ui` library, make the change in the shared-ui library instead of one off in the view component
- Make sure to add a new route for the view
- Make sure to follow all guidelines in `./common-command-guidelines.md` (relative from this path)
